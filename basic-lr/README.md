## 一些有用的信息
- 逻辑回归的损失函数是 log(1+exp(-y*(wT x)))
- 使用简单的梯度下降求解即可
- 数据集可以使用 iris 数据集，iris数据集有3个类别，请将第1类作为正例，其他类别作为负例，原则上可以实现100%准确率。
- 如果用python的话，可以利用sklearn.dataset.load_iris()直接加载数据集
